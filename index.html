Meta広告 代理店運用
完全セットアップマニュアル

〜 アカウント作成から管理設計まで 〜

全体フロー概要

Meta広告を代理店として複数クライアント分を運用するには、以下の5つのSTEPで環境を整えます。

  1. 1自社のビジネスマネージャー（BM）を作成する
  2. 2クライアントのBMを作成・設定する
  3. 3クライアントのBMにパートナーとして入り、広告アカウント・Pixel・ページを設定する
  4. 4効率よく管理するための仕組みを設計する
  5. 5広告の運用を開始する

大原則として、クライアントのInstagramから直接広告を出してはいけません。 必ずビジネスマネージャーを通じて管理します。BMを使うことで、複数クライアントを安全・効率的に一元管理できます。

1自社ビジネスマネージャーの作成

ビジネスマネージャーとは

ビジネスマネージャー（BM）は、Meta（Facebook・Instagram）の広告・ページ・Pixelなどを一元管理するための無料の管理ツールです。
個人のFacebookアカウントで管理すると、ビジネスと個人が混在してしまい、担当者交代にも対応できず、複数クライアントの管理が困難になります。BMを使うことでビジネスと個人を完全に分離し、複数人での権限管理や、パートナー機能による安全な連携が可能になります。

作成手順

  1. ブラウザで https://business.facebook.com にアクセスします。
    ※BMの作成には個人のFacebookアカウントが必要なので、まだない場合は先にfacebook.comで作成してください。
  2. 右上の「アカウントを作成」をクリックします。
  3. ビジネス名（例：株式会社〇〇）・あなたの氏名・会社のメールアドレスを入力して「次へ」を押します。（なければ個人メールアドレスでも可能です）
  4. 確認メールが届いたら「確認する」をクリックします。ダッシュボードが表示されれば完了です。

2クライアントビジネスマネージャーの作成

クライアントのBMがない場合：新規作成を依頼する

クライアントに https://business.facebook.com にアクセスしてもらい、「アカウントを作成」からビジネスマネージャーを作成するよう依頼します。

⚠️ 重要：FacebookアカウントはInstagramアカウントでは代替できません

ビジネスマネージャーへのログインにはFacebookアカウントが必須です。「Instagram広告だけ出したい」という場合でも、広告を管理・配信する仕組み自体がMetaのビジネスマネージャー上で動いているため、Facebookアカウントが必要になります。

支払い方法（クレジットカード）の登録

クライアントに以下の手順で支払い方法を登録してもらいます。

  1. https://business.facebook.com にアクセスしてログイン
  2. 左上「≡（三本線）メニュー」→「広告マネージャー」を開く
  3. 画面左上「≡」→「請求と支払い」を選ぶ
  4. 「支払い方法を追加」ボタンをクリック
  5. 「クレジットカードまたはデビットカード」を選択
  6. 以下のカード情報を入力して「続行」を押す
    • カード番号
    • 有効期限
    • セキュリティコード（カード裏面の3桁）
    • カード名義人（ローマ字）
    • 請求先住所
  7. カードが一覧に表示されれば登録完了

⚠️ 支払いに関する注意点

  • 広告費は一定金額に達するたびに自動で引き落とされます。残高不足にならないよう注意してください。
  • カード情報を変更した場合は必ず弊社にご連絡ください。カードが使えなくなると広告が自動停止されます。
  • 登録できるのはVisa・Mastercard・American Expressなどの国際ブランドのカードです。一部のデビットカードやプリペイドカードは登録できない場合があります。

3パートナー連携と広告アカウント設定

Meta広告を出稿するには2つの設定が必要です。それぞれクライアント側の作業と弊社側の作業に分かれます。

【クライアントにやってもらう作業】

① Facebookページの連携

FacebookページはクライアントのログインIDとパスワードが必要なため、クライアント自身に行ってもらいます。

  1. https://business.facebook.com にアクセスしてログイン
  2. 左上「≡（三本線）メニュー」→「ビジネス設定」を開く
  3. 左メニュー「アカウント」→「ページ」をクリック
  4. 「＋追加」→「ページを追加」を選ぶ
  5. 自社のFacebookページ名またはURLを入力して追加する
💡 Facebookページをまだ持っていない場合： facebook.com/pages/create にアクセスして作成してください。会社名・業種カテゴリ・ロゴ・住所・電話番号・営業時間を入力しておくと広告の信頼性が上がります。

② Instagramアカウントの連携

InstagramのIDとパスワードが必要なため、クライアント自身に行ってもらいます。

  1. https://business.facebook.com にアクセスしてログイン
  2. 左上「≡（三本線）メニュー」→「ビジネス設定」を開く
  3. 左メニュー「アカウント」→「Instagramアカウント」をクリック
  4. 「＋追加」ボタンをクリック
  5. 会社のInstagramのユーザー名とパスワードを入力してログイン
  6. 一覧にInstagramアカウントが表示されれば連携完了
💡 Instagramがビジネスアカウントになっていない場合： Instagramアプリで「設定 → アカウント → プロアカウントに切り替える」から変更してください。これをしておくと連携がスムーズです。

【クライアントと弊社の連携作業】

弊社側からアクセスリクエストを送るため、まずクライアントに「広告アカウントID（15桁の数字）」を教えてもらう必要があります。

【クライアントへの案内】広告アカウントIDの確認方法

  1. https://business.facebook.com にアクセスしてログイン
  2. 左上の「≡（三本線）メニュー」をクリック
  3. 「ビジネス設定」を選ぶ
  4. 左メニューの「アカウント」→「広告アカウント」をクリック
  5. 画面中央に広告アカウント名と、その下に表示される「広告アカウントID：XXXXXXXXXXXXXXX（15桁）」をそのままコピーして弊社に送ってください。

【弊社側の作業】リクエストを送る手順

  1. 自社のビジネスマネージャーを開く
  2. 「ビジネス設定」→「アカウント」→「広告アカウント」へ進む
  3. 「追加」→「アクセスをリクエスト」を選ぶ（※「新しいアカウントを作成」ではないので注意）
  4. クライアントから教えてもらった広告アカウントID（15桁）を入力してリクエストを送る
  5. クライアントのビジネスマネージャーに通知が届くので、承認ボタンを押して完了

広告アカウントの作成・設定（※弊社で新規作成する場合）

アクセスリクエストではなく、弊社のビジネスマネージャーからクライアント用の新しい広告アカウントを作成する場合の手順です。

  1. 自社のビジネスマネージャーを開く
  2. 「ビジネス設定」→「アカウント」→「広告アカウント」へ進む
  3. 「＋追加」→「新しい広告アカウントを作成」を選ぶ
  4. 以下を設定して作成する
    • 広告アカウント名：命名規則に従って入力（例：ABC社_整体_202504）
    • タイムゾーン：Asia/Tokyo（日本標準時）
    • 通貨：JPY（日本円）
  5. 作成後、クライアントのビジネスマネージャーに広告アカウントを紐付ける

⚠️ 広告アカウント作成時の注意点

  • タイムゾーンと通貨は後から変更できません。 必ず作成時に正しく設定してください。
  • 広告アカウントは1クライアントにつき1アカウントが原則です。 複数クライアントを1アカウントで運用すると、Pixelデータや予算・請求が混在して管理不能になります。

4管理の効率化設計

ここが最も重要なステップです。最初にルールと仕組みを整えておかないと、クライアント数が増えた時点で管理が破綻します。

1. 命名規則を統一する

全クライアントで共通の命名規則を設定することで、担当者が変わっても何を見ているか瞬時に分かります。

  • 広告アカウント名：「クライアント名_業種_開始年月」
    （例：ABC社_EC_202504）
  • キャンペーン名：「[目的][施策名][開始日]」
    （例：CV_春季セール_20250401）
  • 広告セット名：「[ターゲット][配置][入札戦略]」
    （例：30代女性_自動配置_最小CPA）
  • 広告名：「[クリエイティブ種別][訴求][番号]」
    （例：動画_新商品紹介_001）

2. 担当者ごとに権限を分ける

自社ビジネスマネージャーの「設定 → ユーザー → ユーザーを追加」から、スタッフに必要なクライアントの広告アカウントだけを割り当てます。

  • 管理者： 全クライアントにアクセスできる権限
  • 上級担当者： 担当クライアントの全操作ができる広告アカウント管理者権限
  • 一般担当者： 担当クライアントの閲覧・レポートのみのアナリスト権限

3. 管理スプレッドシートを作成する

クライアント数が増えても管理できるよう、全クライアントの状況を一覧できるマスター管理表が不可欠です。スプレッドシートで一元管理することで、どのクライアントがどの状態にあるかを瞬時に把握できます。

【管理推奨項目】

クライアント名 BM所有者 広告アカウントID 社内担当者 契約開始日 月間広告予算 支払い方法 請求サイクル クライアントのBM ID 運用ステータス 備考

5広告キャンペーンの作成

環境が整ったら広告を作成します。Meta広告の構造は「キャンペーン → 広告セット → 広告」の3層になっています。

▶キャンペーン目的の選び方

目的は大きく「認知・検討・コンバージョン」の3カテゴリに分かれます。

  • 新規ブランドの認知拡大： リーチ・ブランド認知度向上
  • サイト訪問を増やしたい： トラフィック
  • 問い合わせや登録を増やしたい： リード獲得
  • EC・予約・申し込みを増やしたい： 売上（購入）

※ほとんどのクライアントではコンバージョン目的（売上やリード獲得など）が中心になります。

▶広告セットの設定

  • ターゲティング： 地域・年齢・性別などの基本設定に加え、興味関心・行動による詳細ターゲット、自社の顧客リストや過去のサイト訪問者を使ったカスタムオーディエンス、既存顧客に似たユーザーへ拡張する類似オーディエンスが使えます。
  • 予算： 日予算（1日あたりの上限）か通算予算（期間全体の上限）を選べます。運用初期やテスト段階では日予算が管理しやすいです。

▶広告（クリエイティブ）の作成

広告マネージャーで対象キャンペーンを選び「+作成」から進みます。

  1. フォーマット選択：
    • 1枚の画像または動画（最もよく使う）
    • カルーセル（複数商品の訴求に有効）
    • コレクション（ECサイト向け）
  2. 入稿作業： Facebookページ・Instagramアカウントが正しく連携されているか確認してから、画像・動画・テキスト・CTA（コールトゥアクション）ボタンを入力します。
  3. 公開： プレビューを確認して公開します。
